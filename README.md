Employee Management System (Java OOP Assignment)
Overview

This project is a Java-based Employee Management System developed using Object-Oriented Programming (OOP) concepts.
The system allows users to add employees, view employee details, search employees by ID, and process payroll.

The program demonstrates the use of abstraction, inheritance, interfaces, encapsulation, and polymorphism in Java.

Features

The system provides the following functionalities:

Add Employee

Add Full-Time employees

Add Part-Time employees

View All Employees

Displays all employee details stored in the system

Search Employee by ID

Find a specific employee using their ID

Process Payroll

Calculates employee bonus

Displays final salary after bonus

Exit

Safely exits the application
